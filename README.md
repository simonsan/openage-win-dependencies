# openage-win-dependencies
Prebuild x86/x64 windows dependencies

The [package](https://github.com/simonsan/openage-win-dependencies/releases) contains prebuilt versions of these libraries:

 - dirent
 - eigen3
 - fontconfig
 - freetype
 - harfbuzz
 - libepoxy
 - libogg
 - libpng
 - opus
 - opusfile
 - sdl2
 - sdl2-image
